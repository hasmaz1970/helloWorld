    SC-02: Topup Balance of a Bucket

    SC-02.01: Basic Topup

    Request

    POST https://monetizehub.io/public/pbal/accountBalanceManagement/v4/topupBalance
Content-Type: application/json

{
  "amount": {
    "amount": 20
  },
  "bucket": {
    "id": "<<BA-01-spendingWallet>>"
  },
  "reason": "rechargeDueTo_BizReason_X",
  "channel": {
    "name": "<<refOnly>>",
    "id": "<<refOnly>>"
  },
  "requestor": {
    "name": "<<refOnly>>",
    "id": "<<refOnly>>"
  }
}

    Response

    Compliant to tmf654.

    201 Created

{
  "id": "<<topUpRequestId-01>>",
  "status": "created|completed"
  ...
}
